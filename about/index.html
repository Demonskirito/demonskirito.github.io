关于我

你好，我是念云

一位编程/ACGN/摄影/剪辑的爱好者

编程

  • 常见的编程语言几乎都会，至少使用过
    • 一般都在做全栈的开发，使用过的前端框架有 vue、uni-app，后端使用过 flask、springboot
    • 目前常用的是  java，其他编程语言已较少接触
    • 因为对小程序很感兴趣，所以即用过微信的原生格式代码写，也用过 uni-app 写，对小程序的发布和代码规范很熟练
    • 数据库常用的是 mysql 和 readis
    • 为了能将页面在公网上展示，所以去熟练掌握了宝塔面板的操作和 Linux 的使用
    • 因为自己现在的专业是数据科学与大数据技术，所以学习了一点大数据相关的技术栈，Hadoop、hive、spark、python….
  • 因为对算法感兴趣，打过一段时间的 codeforce 和力扣
  • 平时喜欢研究一些插件的使用，比如最近和同学刚完成一个利用 CSDN 图床上传图片获取 url 的软件 exe，方法和绝大多数市面上的一些插件相同，写 python 的脚本再利用 python 特有的打包方法 Pyinstaller 实现，但由于最近写博客发现 CSDN 的图床不支持外链，所以只能用用 url，其它格式就不行，例如 markdown
  • 后端 api 文档（smart-doc）的使用和 vue 打包 electron 插件的使用就不细说了

ACGN

ACGN 方面了解的还算多，但现在已经不会太投入了。

Anime

  • 看过的动漫非常多，已经点不清了，喜欢的动漫有《死亡笔记》《命运石之门》《反叛的鲁路修》《东京地震 8.0》《从零开始的异世界生活》《天元突破 红莲螺岩》《86-不存在的战区》《夏目友人帐》《东京喰种》《我的青春恋爱物语果然有问题》……有点多就不说了，只看自己感兴趣的
  • 入宅番是《火影忍者》《龙珠 Z》《刀剑神域》《天降之物》和《约会大作战》
  • 看的最久的是《妖精的尾巴》《火影忍者》《龙珠 Z》
  • 最近的一次是在看《东京复仇者》和《强风吹拂》
  • 虽然已经很少去看新番了，但还是会感慨动漫对我人生带来的影响和改变

Comic

  • 正经漫画看的比较少，最让我震撼的动漫果然还是日本的青年漫《剑风传奇》《孤高之人》《浪客行》《冰海战记》
  • 这几年上了大学才开始认认真真的去看漫画，有《有花无实》《我推的孩子》《少年的深渊》《青之芦苇》《地狱乐》
  • 看过《女朋友、借我一下》，结果因为剧情太烂，中途弃坑了

Game

  • 主要玩电脑游戏
    • Steam 游戏
      • 玩的最久的是 CS，非常喜欢
      • 魂系游戏最喜欢玩老头环，虽然今年的黑神话悟空也挺好玩的，但总觉得差点意思
    • 其他游戏
      • 米哈游的游戏，不捧不黑，基本只领邮件抽卡，做做每日
  • 手游
    • 腾讯游戏
      • 火影忍者、王者荣耀都玩的还行

Novel

  • 几乎不怎么看小说
  • 破灭文学小说爱好者
    • 最近看完的一本书叫《窄门》，法国文学，爱情的“窄门”很不好走
    • 最喜欢的作品是《苦役列车》和《活着》，《人间失格》和《山月记》也很好看
    • 最近在看谷崎润一郎的《初期短篇集》

Music

  • 重度音乐患者，已经听了 4500 个小时了
  • 喜欢的音乐风格有很多，一般只在网易云听歌
  • 2024 年最喜欢的音乐
    • 《The Hardest Part》
    • 《dont let me go》
    • 《God’s Word》
    • 《READLINE DAY》
  • 其实还有许多喜欢听的歌，但今年这些歌可能对于我来说更具有意义吧

摄影

因为我喜欢出去玩，并且喜欢用手机去拍照，后来就接触到了相机，通过相机我将许许多多难忘的回忆封存在了照片里，为此我还特意去学习了 lr 和 ps，虽然不怎么拍人像，但风景像拍出来的照片，我还是非常喜欢的，但还没有达到自己想要的水平，所以照片并没有存放在太过正式的平台，后面可能会放在博客展示吧。
